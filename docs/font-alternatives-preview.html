Font-Alternativen – Live-Vorschau

Diese Datei lädt unten zusätzlich Schriften von Google Fonts zum Vergleich. Zum Ansehen die Datei im Browser öffnen (Doppelklick oder open docs/font-alternatives-preview.html). System-Stacks (erster Block) brauchen kein Google Fonts – nur lokal installierte Schriften. UI Rounded in der App: zuerst SF Pro Rounded (lokal), dann ui-rounded (nur Safari) – Chrome ignoriert ui-rounded, nutzt daher die Namens-Schrift oder System-Sans.
